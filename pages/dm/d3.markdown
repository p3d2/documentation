## 3. Name convention

Use different tags to describe files content when you name files; if name is automatically generated by some software, do not forget to clearly describe data in the readme.

Possible tags to name files:

- Date YYYYMMDD
- Experiment name
- Method
- Project ID
- Version
- Subset
- Source ID
- ...
